The Run Hub

View site >>

A website I designed and developed for The Run Hub, a company which specializes in all things related to triathlons and running. The Run Hub is a one-stop online retailer for everything running: running shoes, running apparel, accessories and more. Whether you run for fun or competitively they will cater for you. Going the extra mile for customers is the foundation on which the business is built. It is a hub for runners catering for every running requirement imaginable. Rather than just being a retail store, they offer additional services such as coporate talks, race sponssorship, mobile shops at club nights or event and coaching packages.

Project Overview





The Run Hub found itself in need of a new, refreshed look that will represent the brand in a more appropriate and impressive way. Our goal was to keep the interface intuitive and the user experience smooth and satisfying, as well as present the aesthetic in a way that reflects the brand mission and lifestyle that the company promotes.





Design Process

Redesigning an existing site is always a challenge since it is important to maintain the look and feel that users associate with a brand.

1. Review & Research
First, we conducted a heuristical analysis to define weakspots of the site. Using Google Analystics, we were also able to identify where people were dropping off. This helped us put together a strategy to improve the clients conversion rates.
2. Wireframing
After presenting the initial UX review to the clients, and gaining a shared understanding of the direction we needed to go in, we created wireframes to establish how the site would be laid out, navigation structure and how key pieces of functionality would work.
3. Design
The goal was to provide an aesthetic upgrade for the existing store, making use of beautiful imagery and suitable textures. How we could enhance the brand image, as well as improve store functionality were both considered.
4. Development
During all steps of the process, we were constantly communicating with the client, receiving all the feedback we needed after each iteration. Once the design was agreed on, we developed the site using HTML, CSS, JQuery, and NitroScript.


Personas

Based on the research found in the discovery period of the project a persona was developed, Shauna. Designing for Shauna helped in making informed design decisions as I had the end user always in mind.


Storyboarding

Keeping Shauna in mind, I story boarded the userâ€™s behavioral flow from homepage through various scenarios on the current RunHub site. These flows helped in defining the necessary features required for Shauna to successfully navigate through the site.



Wireframing

A progression of key screens from low fidelity wireframes which were shared with the client through Invision and were annotated to describe some of the key interactions. These wireframes were iterated on and were eventually fully approved by the client.



Style Guide

Created early in the design process, this style guide was useful for keeping me on brand at all times.



End Result

The newly designed site was a leap forward for The RunHub. It was a design created with the user in mind. Amongst its key features were: improved mega-menu navigation, custom product carousels, promotional categories on the homepage, live-chat, Instagram feed, gait analysis info page, a clear and effective FAQ page, fully responsive and beautiful on all devices.

The RunHub has seen a significant increase in it's conversion rates as a result of all these improvements. As well as a decrease in the amount of frustrated customers ringing with questions about delivery, and shoe sizes

  • Date: October 2018
  • Client: The Run Hub
  • Category: Design & Development

eBay Integration


NitroSell is an eCommerce company which specializes in integrated eCommerce, I was part of the team which worked on a new eBay integration. The goal was to create a solution which would integrate well with a retailers existing Point Of Sale (POS) data.



Project Overview





I was responsible for several key parts of the eBay integration. I created user flows, wireframes, as well as creating the front-end using HTML, CSS, JQuery & Bootstrap. Below are the outlines and results of some of these processes.



Information Architecture

I created a content taxonomy map, to give meaning to the defined features, and a site map to organize the pages, information, and navigation of the application. This was based off of feedback gained through user interviews and card-sorting exercises conducted with customers and internal stakeholders.



Wireframes

Wireframes were creates for key screens. These wireframes were used by other members of the team for user testing purposes. Once feedback was received from these tests, we iterated and improved the wireframes.



Development

Below are some of the key screens which i developed.

On this project, I worked closely with key stakeholders, such as product owners, scrum team managers and individual dev members to gain a deeper understanding of what was needed. I created and presented wireframes, did competitor analysis and iterated the design until the user journey was at a very high level, all the functionality was accounted for, and the stakeholders were happy. I was also responsible for building the front-end of the eBay integration, using HTML, CSS, Bootstrap and JQuery.
